# Rayeva AI Systems

A production-ready MERN stack application with AI-powered product categorization and sustainable B2B proposal generation.

## Features

### Module 1: AI Auto-Category & Tag Generator
- Automatic product categorization from predefined list
- AI-powered subcategory suggestions
- SEO-optimized tag generation (5-10 tags)
- Sustainability filter recommendations
- Structured JSON storage in MongoDB
- Complete AI prompt and response logging

### Module 2: AI B2B Proposal Generator
- Sustainable product mix optimization within budget
- Quantity and unit cost recommendations
- Automatic total cost calculation and validation
- Environmental impact summary generation
- Proposal storage and management
- Real-time budget tracking

## Tech Stack

### Backend
- Node.js with Express
- MongoDB with Mongoose
- gemini api key
- Joi validation
- Modular architecture (controllers, services, models)

### Frontend
- React with Vite
- TailwindCSS v4
- Framer Motion animations
- Glassmorphism UI design
- Responsive mobile-first design
- React Hook Form
